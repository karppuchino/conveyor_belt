# Conveyor System - Älykäs kuljetinhihnalajittelu

## Project description - Projektin kuvaus

This project is a simulation of a smart conveyor belt in a logistics center implemented in the CodeSys environment. The system recognizes products, sorts them into three different bins based on size, and collects production data.

Tämä projekti on CodeSys-ympäristössä toteutettu simulaatio logistiikkakeskuksen älykkäästä kuljetinhihnasta. Järjestelmä tunnistaa tuotteet, lajittelee ne koon perusteella kolmeen eri lokeroon ja kerää tuotantodataa.


## Technologies -Teknologiat
- **CodeSys V3.5** - PLC programming
- **Structured Text (ST)** -Programming language
- **CodeSys Visualization** - HMI-user interface
  
## Functionalities - Toiminnallisuudet
- Controlling the conveyor belt
- Automated generation of products/packages
- Sorting packages into 3 categories (small/medium/large)
- Counters and indicator lights
- HMI-user interface
  
## Installation - Asennus ja ajaminen
1. Download and install Codesys
2. Open the project file
3. Login -> Start
4. Start the conveyor belt and watch the simulation.
